# Dockerized APIs


- Go through the process in today's lesson to dockerize your API.  Once you can successfully build and run your container locally, push it up to Dockerhub.

- When your image is pushed to Dockerhub, post your image details, including a brief description, in Slack.

- Browse through your classmate's images and pick one you'd like to try out.

Note: You might run into some issues if your local machine has a different architecture than someone else's. You can start [here](https://docs.docker.com/engine/reference/commandline/buildx_build/) under the `--platform` flag.

- Write a frontend that consumes both your own dockerized API and your classmate's.
